## Nouvelles

* **Voir les n derniers commits**:
  ```
  $ git log -n
  ```

* **Enregister les fichiers ajoutés sur le dernier commit**:
  ```
  $ git commit --amend
  ```
  Attention, en cas de branche déjà pousée sur github, la synchronisation doit être forcée :
  ```
  $ git push -f origin mabranche
  ```
